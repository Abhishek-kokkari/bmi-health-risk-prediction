# 🏥 BMI Health Risk Prediction

## 🚀 Live Demo
https://bmi-health-risk-prediction-6125.streamlit.app/

---

## 📌 Project Overview
This project is a machine learning-based web application that predicts health risk based on BMI (Body Mass Index). It helps users understand their health condition using simple inputs.

---

## 🛠️ Technologies Used
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit  

---

## 📊 Features
- User-friendly web interface  
- Real-time prediction  
- Data visualization  
- Health risk classification  

---

## ⚙️ How It Works
1. User enters input details  
2. Data is processed and cleaned  
3. Machine learning model predicts health risk  
4. Results are displayed instantly  

---

## 📁 Project Structure
- app.py → Streamlit app  
- models/ → ML model files  
- data/ → Dataset  
- requirements.txt → Dependencies
